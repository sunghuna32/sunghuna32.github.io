> Mac을 사용하는 데 유용한 Tip 들을 정리한다.

### 맥 기본 버튼
- ⌥   : 옵션_option 키
- ⌘   : 커맨드_command 키
- ⇧, ⇪: 시프트_shift 및 대문자 잠금_capslock 키
- ⌃   : 컨트롤_control 키
- ⇥   : 탭_tab 키
- ⎋   : 에스케이프(ESC) 키

### page & move home (end)
- fn + ↑ : Page Up
- fn + ↓ : Page Down   
- fn + ← : Home  
- fn + → : End
